Preparar emtorno para instalar kivy

--------python -m pip install --upgrade pip setuptools virtualenv


Crear entorno Virtual de KivyDM

--------python -m virtualenv "nombre del entorno virtual"


Activar el entorno cada vez que se abra la terminal de python.

-Windows CMD

---------kivy_venv\Scripts\activate

.Windows bash terminal

---------source kivy_venv/Scripts/activate

-Linux o MAC 

---------source kivy_venv/bin/activate
